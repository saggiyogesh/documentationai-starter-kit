## Recent Updates

<Callout kind="info">
Stay current with the latest enhancements to Yogesh Agrawal's documentation platform. Each release includes new features, performance improvements, and bug fixes to enhance your documentation experience.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors in themes, including `#3B82F6` as the default blue.
- Introduced real-time collaboration previews for documentation editing.
- Enhanced search functionality with fuzzy matching and syntax highlighting.

## Improvements

- Optimized page load times by `>30%` through better asset bundling.
- Improved mobile responsiveness for all documentation pages.

## Bug Fixes

- Fixed anchor link navigation issues in long documents.
- Resolved table rendering glitches on certain browsers.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated MDX component library with 15+ interactive elements like Steps and Tabs.
- Added changelog tracking with Update components for version history.

## Bug Fixes

- Corrected frontmatter parsing errors in YAML blocks.
- Fixed escaping issues for special characters like `{` and `<` in prose.

## Improvements

- Streamlined component syntax rules to prevent common parsing failures.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Initial Release

- Launched core documentation platform with MDX support.
- Included essential components: Callout, Cards, CodeGroup, and more.
- Established consistent API patterns using `https://api.example.com` endpoints.

## Breaking Changes

- Updated attribute syntax to require double quotes for all strings.

</Update>
