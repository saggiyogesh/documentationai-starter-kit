## Overview

Welcome to Yogesh Agrawal's documentation platform. You can organize, manage, and share all your project documentation in one centralized space. This platform streamlines workflows, making it easy to create, collaborate on, and maintain high-quality docs for any project.

## Key Benefits

Yogesh Agrawal documentation offers powerful tools tailored for project management. You benefit from structured organization that keeps information accessible, version control to track changes, and seamless search capabilities to find content quickly.

## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation page.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create your account using your email.
  </Step>
  <Step title="Create Workspace" icon="home">
    Set up a new workspace for your project. Name it after your initiative, like "Project Alpha".
  </Step>
  <Step title="Add First Page" icon="plus">
    Click the "New Page" button and start writing in Markdown or rich text editor.
  </Step>
  <Step title="Publish" icon="globe">
    Save and publish your page. Share the link with your team.
  </Step>
</Steps>

<CodeGroup tabs="Markdown,Rich Text">
  ```markdown
  # My First Doc

  Welcome to the team!

  - [ ] Task 1
  - [ ] Task 2
  ```
  ```html
  <h1>My First Doc</h1>

  <p>Welcome to the team!</p>

  <ul>
    <li><input type="checkbox"> Task 1</li>
    <li><input type="checkbox"> Task 2</li>
  </ul>
  ```
</CodeGroup>

## Navigation Basics

Navigate the platform effortlessly. Use the sidebar for quick access to workspaces, the search bar for content discovery, and the top menu for account settings.

<Tabs>
  <Tab title="Sidebar" icon="menu">
    Access recent pages, workspaces, and settings from the left sidebar. Pin frequently used docs for faster navigation.
  </Tab>
  <Tab title="Search" icon="search">
    Type keywords into the global search to find pages, sections, or even code snippets across your documentation.
  </Tab>
  <Tab title="Editor" icon="edit-3">
    Switch between Markdown and WYSIWYG modes. Preview changes live before publishing.
  </Tab>
</Tabs>
