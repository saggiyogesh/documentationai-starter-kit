---
title: Cannot Access Documentation
description: Steps to fix access issues on Yogesh Agrawal's docs platform.
---

## Symptoms

- Documentation pages fail to load
- "Access Denied" error

## Solution

<Steps>
  <Step title="Check login status">
    Make sure you are logged in at `https://docs.yogeshagrawal.com/login`.
  </Step>
  <Step title="Verify permissions">
    Confirm you have access to the specific project. Ask project owner for an invite if needed.
  </Step>
  <Step title="Clear cache">
    Try clearing browser cache and refresh the page.
  </Step>
</Steps>
