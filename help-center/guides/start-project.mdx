## Create Your First Project

<Steps>
  <Step title="Log in">
    Visit `https://docs.yogeshagrawal.com` and sign in.
  </Step>
  <Step title="Create project">
    Click "New Project" and enter a name for your documentation space.
  </Step>
  <Step title="Add pages">
    Use the interface to add new pages: README, Setup, API, etc.
  </Step>
  <Step title="Publish">
    Click "Publish" to make your docs live.
  </Step>
</Steps>
